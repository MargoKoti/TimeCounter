# TimeCounter
HTML5/CSS3/bootstrap/fontawesome/JS

Program umożliwiający odliczanie czasu do ważnego wydarzenia.
Poza możliwością ustawienia docelowej daty, umożliwia dodanie przez użytkownika dowolnego zdjęcia.
